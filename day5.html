Data Types & Variables in Java

Understanding data types, variables, and how information is stored in Java memory.

Data Types Variables Primitive Types Memory int float double char boolean
🔁
Quick Revision (Day 4)

Java = Object-Oriented Programming Language

  • World is a collection of objects
  • Every object belongs to a class
  • Every object has: State (HAS) → Data & Behavior (DOES) → Methods

Key Concepts:

  • Class = Blueprint (imaginary)
  • Object = Real entity
From Day 4: Objects have State (properties) and Behavior (methods). Today we dive deeper into how State is stored using Data Types.

Today's Focus
Now we move into actual Java coding foundation

Topics for Today:

  • Data Types
  • Variables
  • How data is stored in Java
🧠
What is Data?
Data
Any information that represents facts or values

Examples of Data:

  • Name → "Swathi"
  • Age → 21
  • Marks → 95.5
  • Gender → 'F'
In programming: Everything we store = Data
📦
What is a Data Type?
Data Type
Type of data a variable can store. It tells the computer what kind of value you are storing.

Why Data Types?

Computer memory is limited, so it needs to know:

  • How much space to allocate
  • What type of value is stored
🔢
Types of Data in Java
Java mainly has 2 categories of Data Types

🔹 Primitive Data Types

Built-in, basic data types

Data Type Example Description
int 10 Integer numbers
float 10.5f Decimal numbers
double 10.555 More precise decimal
char 'A' Single character
boolean true/false Logical values

🔹 Non-Primitive Data Types

Created by programmer. Examples:

  • String
  • Arrays
  • Classes
  • Objects
🔤
Important Primitive Data Types

🔸 int (Integer)

Stores whole numbers

int age = 21;

🔸 float

Stores decimal numbers (less precision)

float price = 99.5f;

🔸 double

Stores decimal numbers (high precision)

double salary = 50000.75;

🔸 char (Character)

Stores single character

char grade = 'A';

🔸 boolean

Stores true/false (logical values)

boolean isPassed = true;
🧾
What is a Variable?
Variable
A named memory location used to store data. Think of it as a container to hold information.

Real-Life Analogy:

📦 Box = Variable
→
📝 Item inside = Data
✍️
Syntax of Variable

Basic Syntax:

dataType variableName = value;

Examples:

int age = 21;
String name = "Swathi";
float marks = 95.5f;
boolean isActive = true;
char gender = 'F';
⚠️
Rules for Naming Variables

✅ Rules to Follow:

  • Must start with letter, underscore (_), or dollar ($)
  • Cannot start with number ❌
  • No spaces ❌
  • Case-sensitive (age ≠ Age)

✅ Valid Names:

int age;
int _count;
int $price;
int myAge;

❌ Invalid Names:

int 1age;
int my age;
int my-value;
int class;
🧠
Memory Understanding (VERY IMPORTANT)
When you write: int age = 21;

What Happens Behind the Scenes:

  • Memory is allocated for variable "age"
  • Value 21 is stored in that memory
  • Variable "age" is a reference to that memory location
1. Declare
→
2. Allocate Memory
→
3. Store Value
→
4. Reference

Connection with OOP
From Day 4: Objects have properties (State). Properties are stored using Data Types and Variables.

Example from Day 4:

class Car {
    String name;
    int cost;
    float mileage;
}

Here:

  • name, cost, mileage = State (HAS part)
  • Data types define what they store
  • Variables hold the actual values

  • Non-primitive types = Complex/advanced types
  • Variables + Data Types = Foundation of Java
