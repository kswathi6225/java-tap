Introduction to Arrays – The Problem
❌ Problem with Old Method
Before arrays, storing data like this:
int a, b, c, d, e;

For 5 students: OK ✔

For 50 students: Messy ❌

🚨 Issues:
✔ Too many variables
✔ Hard to manage
✔ Difficult to retrieve data

💡 Solution → ARRAYS

  • ✔ Storage problem solved
  • ✔ Retrieval problem solved
  • ✔ Code readability improved
🚀
What is an Array?
Definition:
Array = Collection of similar (homogeneous) data

Example:

int[] ages = new int[5];

👉 Stores 5 integer values
🔥
Key Characteristics
  • ✔ Stores same type of data
  • ✔ Stored in continuous memory
  • ✔ Access using index
  • ✔ Arrays are objects in Java
🚀
Important Concept – Index
📌 Index always starts from 0
Student Index
1st 0
2nd 1
3rd 2
4th 3
5th 4
💡 Formula: nth element → index = n - 1
🔥
Steps to Create Array (VERY IMPORTANT)

✔ 3 Key Observations:

  • 1️⃣ Dimensionality – 1D, 2D, 3D
  • 2️⃣ Type of Data – Homogeneous (same type)
  • 3️⃣ Nature – Regular (for now)
🚀
Creating 1D Array

Syntax:

int[] a = new int[5];
Part Meaning
new Calls JVM
int Data type
[5] Size
a Reference
🔥
Memory Representation
👉 JVM creates array in Heap Memory

Example:

int[] a = new int[5];

Internally:
Index:   0   1   2   3   4
Value:   0   0   0   0   0
💡 Important:
✔ Default value of int → 0
✔ Index starts from 0
🚀
Accessing Elements 
System.out.println(a[2]);

👉 Access 3rd element
🔥
Updating Values 
a[0] = 10;
a[1] = 20;
a[2] = 30;
a[3] = 40;
a[4] = 50;
🚀
Input Using Scanner 
import java.util.Scanner;

Scanner sc = new Scanner(System.in);

for(int i = 0; i < 5; i++) {
    a[i] = sc.nextInt();
}
🔥
Printing Array 
for(int i = 0; i < 5; i++) {
    System.out.println(a[i]);
}
🚀
Why Loops are Important
❌ Without loop:
a[0] = sc.nextInt();
a[1] = sc.nextInt();
...
✔ With loop:
Avoids repetition ✔
Cleaner code ✔
Scalable ✔
🔥
2D Array (Matrix Concept)

📌 Scenario:

  • 👉 2 classrooms, 5 students each

Syntax:

int[][] a = new int[2][5];
Dimension Meaning
2 Rows (classrooms)
5 Columns (students)
🚀
Accessing 2D Array 
a[1][0] = 20;

👉 Row 1 → 2nd classroom
👉 Column 0 → 1st student
🔥
3D Array

📌 Scenario:

  • 👉 2 schools → 3 classes → 5 students

Syntax:

int[][][] a = new int[2][3][5];
Level Concept
1 School
2 Classroom
3 Student
🚀
Accessing 3D Array 
a[1][0][2] = 15;

👉 School 2 → Class 1 → Student 3
🔥
Important Notes
  • ✔ Arrays are stored in heap
  • ✔ Default values are assigned
  • ✔ Index starts from 0
  • ✔ Access using [] operator
🚀
Real Insight (VERY IMPORTANT)
👉 In real world:
Most problems use 1D arrays
2D arrays → matrices
3D arrays → rare
💡 Interview Tip: 90% questions → 1D arrays
🔥
Key Concept (VERY IMPORTANT)

✔ Easy Access:

  • Want 323rd student?
  • a[322]
  • 👉 No confusion like variables
✔ Array = collection of same data
✔ Index starts from 0
✔ Use loops for efficiency
✔ 1D, 2D, 3D arrays
✔ Stored in heap
