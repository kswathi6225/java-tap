Type Casting 🔀

Converting between data types – Implicit & Explicit

Data Types
Widening
Narrowing
Conversion
🔥
What is Type Casting?
Definition:
Type casting = Converting one data type into another data type

Also called: Type Conversion

Example:

int a = 10;
double b = a;   // int → double
⚡
Types of Type Casting

There are 2 types:

  • ✅ Implicit Type Casting (Automatic)
  • ❌ Explicit Type Casting (Manual)
🚀
Implicit Type Casting (VERY IMPORTANT)
Definition:
Conversion of smaller data type → larger data type automatically. No manual work needed. Done by Java automatically.

📊 Data Type Order (IMPORTANT):

byte → short → int → long → float → double
            ↘ char ↗

Size increasing → conversion allowed

✅ Examples:

byte a = 10;
int b = a;      // byte → int ✔

int x = 50;
double y = x;   // int → double ✔

🧠 Special Case: char → int

  • Allowed: Yes
  • Why: Java uses Unicode; characters internally stored as numbers 
char ch = 'A';
int val = ch;   // 65
💡
Important Exception (INTERVIEW)

❓ Question: Can we convert long → float?

✔ Answer: YES ✅

👉 Why?

  • long: Uses base-2 format
  • float: Uses IEEE format (compressed storage)
  • Float can store larger range due to special representation
⚠️ But: Precision loss may occur!
🎯
Real Meaning (BEST EXPLANATION)

👉 Think like containers:

Small → Big Result
1L → 2L No overflow
2L → 4L No overflow
✔ Smaller → Larger = SAFE (no data loss)
🧠
Another Name
Implicit type casting
= WIDENING
Data gets wider as it moves to a larger container!
🚫
Explicit Type Casting
Definition:
Converting larger → smaller data type manually

✔ Risk:

  • Data loss ❌
  • Overflow ❌

❌ Wrong Code:

double a = 45.7;
int b = a;   // ERROR

✅ Solution (Manual Casting):

double a = 45.7;
int b = (int) a;   // 45
⚠️ Important: Decimal part will be LOST (45.7 → 45)
🧠
Explicit = Narrowing
Another name:
Explicit type casting = NARROWING
Data gets narrower/squeezed into a smaller container!
📊
Summary Table
Type Direction Automatic Risk
Implicit Small → Big Yes No
Explicit Big → Small No Yes
🔥
Full Example (IMPORTANT) 
byte a = 10;
double b = a;     // implicit

double x = 45.7;
int y = (int)x;   // explicit

System.out.println(b); // 10.0
System.out.println(y); // 45
🎯
Interview Questions (VERY IMPORTANT)

Be ready to answer:

  • What is type casting?
  • Difference between implicit & explicit?
  • What is widening?
  • What is narrowing?
  • Can long → float? (TRICK QUESTION)
  • Why char → int works?
These are VERY common interview questions!
🚀
Extra Important Concept

👉 Java internally:

  • Integer: Uses Base 2 format
  • Float/Double: Uses IEEE format
That's why conversions behave differently!
🔥
Final Quick Revision
  • Type casting = conversion between data types
  • Implicit = small → big (automatic)
  • Explicit = big → small (manual)
  • Widening & Narrowing key terms
  • Precision loss in explicit casting
